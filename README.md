# Phantom Node

Free Windows VPS via AppVeyor with Hermes pre-installed.

## Setup

1. Go to Actions → Deploy → Run workflow
2. Enter your Mimo API key (optional)
3. SSH into the machine using the connection info in CONNECTION.md
4. Run `C:\start.bat`

## Config

- Config: `~/.hermes/config.yaml`
- API Key: `~/.hermes/.env`
